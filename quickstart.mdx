# Quickstart Guide

This guide will help you write and run your first Yoyo program.

<Note>
  Make sure you've completed the [installation](/installation) steps before continuing.
</Note>

## Your First Yoyo Program

Let's start with a simple "Hello, World!" program.

<Steps>
  <Step title="Create a Yoyo source file">
    Create a file called `hello.yoyo` with the following content:
    
    ```yoyo hello.yoyo
    main: fn = {
        "Hello, World!".test::print();
    }
    ```
    
    This program defines a `main` function that prints a greeting.
  </Step>
  
  <Step title="Create a C++ host application">
    Create a C++ file to host the Yoyo engine:
    
    ```cpp main.cpp
    #include <engine.h>
    #include <parser.h>
    #include <fstream>
    #include <sstream>
    
    // Choose your backend
    #ifdef USE_LLVM
    #include <llvm/llvm_engine.h>
    using EngineType = Yoyo::LLVM::LLVMEngine;
    #else
    #include <yvm/yvm_engine.h>
    using EngineType = Yoyo::YVM::YVMEngine;
    #endif
    
    int main() {
        // Read source file
        std::ifstream file("hello.yoyo");
        std::stringstream buffer;
        buffer << file.rdbuf();
        std::string source = buffer.str();
        
        // Create engine
        EngineType engine;
        
        // Parse source code
        Yoyo::Parser parser(source);
        auto statements = parser.parseProgram();
        
        if (parser.failed()) {
            std::cerr << "Parse failed!" << std::endl;
            return 1;
        }
        
        // Add source to engine
        engine.sources["main"] = {"hello.yoyo", std::move(statements)};
        
        // Compile and run
        engine.execute();
        
        return 0;
    }
    ```
  </Step>
  
  <Step title="Compile and run">
    Build your application with Meson or your preferred build system:
    
    ```bash
    g++ -std=c++20 main.cpp -lyoyo -o hello
    ./hello
    ```
    
    You should see:
    ```
    Hello, World!
    ```
  </Step>
</Steps>

## Working with Variables

Yoyo supports type inference with the `:=` operator:

```yoyo
main: fn = {
    // Type inference
    name := "Yoyo";
    version := 1;
    
    // String interpolation
    "Language: ${name} v${version}".test::print();
}
```

## Defining Functions

Functions use the `name: fn = body` syntax:

```yoyo
add: fn (a: i32, b: i32) -> i32 = {
    return a + b;
}

// Single-expression functions can omit braces
square: fn (x: i32) -> i32 = return x * x;

main: fn = {
    result := add(5, 3);
    "5 + 3 = ${result}".test::print();
}
```

## Creating Structs

Yoyo supports structs with methods:

```yoyo
Point: struct = {
    x: i32,
    y: i32,
    
    // Constructor
    new: fn (x: i32, y: i32) -> Point = {
        return Point{ .x = x, .y = y };
    }
    
    // Method
    distance: fn (&this) -> f64 = {
        dx := this.x * this.x;
        dy := this.y * this.y;
        return (dx + dy).sqrt();
    }
}

main: fn = {
    p := Point::new(3, 4);
    "Distance: ${p.distance()}".test::print();
}
```
