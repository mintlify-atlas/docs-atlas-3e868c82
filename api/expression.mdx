## Overview

The Expression module defines all expression types in Yoyo's Abstract Syntax Tree (AST). Each expression class inherits from the base `Expression` class and represents a specific syntactic construct that evaluates to a value.

All expressions participate in type checking and are eventually lowered to the borrow checker's intermediate representation.

## Base Class

### Expression

The abstract base class for all expressions.

```cpp
class Expression : public ASTNode {
public:
    ~Expression() override = default;
    virtual ExpressionVariant toVariant() = 0;
    Type evaluated_type;
};
```

<ParamField path="evaluated_type" type="Type">
  The type of the expression, populated by the type checker
</ParamField>

### Factory Method

```cpp
static auto attachSLAndParent(
    std::unique_ptr<Expression> self, 
    SourceLocation bg, 
    SourceLocation end,
    ASTNode* parent = nullptr
) -> std::unique_ptr<Expression>;
```

<ResponseField name="attachSLAndParent">
  Attaches source location and parent information to an expression
</ResponseField>

## Control Flow Expressions

<Expandable title="IfExpression">

### IfExpression

Represents conditional expressions with `if-else`.

```cpp
class IfExpression : public Expression {
public:
    std::unique_ptr<Expression> condition;
    std::unique_ptr<Expression> then_expr;
    std::unique_ptr<Expression> else_expr;
    bool then_transfers_control = true;
    bool else_transfers_control = true;
    
    IfExpression(std::unique_ptr<Expression> cond, 
                std::unique_ptr<Expression> then_, 
                std::unique_ptr<Expression> else_);
    ExpressionVariant toVariant() override;
};
```

<ParamField path="condition" type="std::unique_ptr<Expression>">
  The boolean condition
</ParamField>

<ParamField path="then_expr" type="std::unique_ptr<Expression>">
  Expression evaluated when condition is true
</ParamField>

<ParamField path="else_expr" type="std::unique_ptr<Expression>">
  Expression evaluated when condition is false
</ParamField>

<ParamField path="then_transfers_control" type="bool">
  True if the then branch returns/breaks/continues
</ParamField>

<ParamField path="else_transfers_control" type="bool">
  True if the else branch returns/breaks/continues
</ParamField>

**Example:**
```yoyo
let max = if a > b { a } else { b };
```

</Expandable>

<Expandable title="ConditionalExtraction">

### ConditionalExtraction

Represents pattern matching with conditional extraction.

```cpp
class ConditionalExtraction : public Expression {
public:
    enum CaptureType { Own, Ref, RefMut };

    CaptureType then_capture_tp;
    CaptureType else_capture_tp;
    std::unique_ptr<Expression> condition;
    std::unique_ptr<Expression> body;
    std::string else_capture;
    std::unique_ptr<Expression> else_body;
    bool then_transfers_control = true;
    bool else_transfers_control = true;
    
    ConditionalExtraction(
        std::string name,
        CaptureType cap,
        std::unique_ptr<Expression> cond,
        std::unique_ptr<Expression> body,
        std::unique_ptr<Expression> else_,
        std::string else_name = "",
        CaptureType else_cap = Own);
    ExpressionVariant toVariant() override;
};
```

<ParamField path="captured_name" type="std::string">
  The name to bind the extracted value to
</ParamField>

<ParamField path="then_capture_tp" type="CaptureType">
  How the then branch captures (Own, Ref, or RefMut)
</ParamField>

<ParamField path="condition" type="std::unique_ptr<Expression>">
  The expression to extract from
</ParamField>

<ParamField path="body" type="std::unique_ptr<Expression>">
  Expression executed when extraction succeeds
</ParamField>

<ParamField path="else_body" type="std::unique_ptr<Expression>">
  Expression executed when extraction fails
</ParamField>

**Example:**

## Ownership Enumeration

```cpp
enum class Ownership { 
    Owning = 0, 
    NonOwning, 
    NonOwningMut 
};
```

Defines how values are captured or passed:

- **Owning**: Value is moved
- **NonOwning**: Immutable reference
- **NonOwningMut**: Mutable reference

## ExpressionVariant

A type-safe variant containing pointers to all expression types:

```cpp
using ExpressionVariant = std::variant<
    IntegerLiteral*,
    BooleanLiteral*,
    TupleLiteral*,
    ArrayLiteral*,
    RealLiteral*,
    StringLiteral*,
    NameExpression*,
    PrefixOperation*,
    BinaryOperation*,
    GroupingExpression*,
    LogicalOperation*,
    PostfixOperation*,
    CallOperation*,
    SubscriptOperation*,
    LambdaExpression*,
    ScopeOperation*,
    ObjectLiteral*,
    NullLiteral*,
    CharLiteral*,
    GenericNameExpression*,
    GCNewExpression*,
    AsExpression*,
    MacroInvocation*,
    SpawnExpression*,
    TryExpression*,
    BlockExpression*,
    IfExpression*,
    ConditionalExtraction*
>;
```

Used for type-safe visitor pattern implementations across the compiler pipeline.
